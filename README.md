# Advanced File System Analyzer

A fast, multi-threaded command-line tool built with C# (.NET) designed to scan directories, optimize storage, and find hidden space-wasters.

## 🚀 Features

* **Top 10 Largest Files:** Instantly scans and lists the biggest files in any directory.
* **Empty Directories Finder:** Locates useless empty folders left behind by uninstalled software.
* **Smart Duplicate Detection:** Uses **SHA-256 hashing** to find identical files, even if they have different names.
* **High Performance:** Powered by the **Task Parallel Library (TPL)** for multi-threaded, lightning-fast disk scanning.
